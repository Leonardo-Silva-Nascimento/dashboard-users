# Sistema de Gestão de Usuários - Varos

Sistema completo de dashboard e gestão de usuários desenvolvido com Next.js 15, Tailwind CSS e Prisma ORM.

## Funcionalidades

- **Dashboard** com métricas e tabela de clientes
- **Filtros** por consultor
- **Gestão completa de usuários** (CRUD)
- **Relacionamento** entre consultores e clientes
- **Validação de formulários** com Zod
- **Server Components** e Server Actions
- **Loading Skeletons** para melhor UX

## Tecnologias

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Prisma ORM
- SQLite (pode ser facilmente trocado para PostgreSQL/MySQL)
